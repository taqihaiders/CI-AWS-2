## AWS CODE ARtifcat Setup ## 

It is used to sore maven dependencies to build Artifact through Maven,
It work as a storage of libraries

Simple Workflow:
Create a repository in AWS CodeArtifact.
Log in to CodeArtifact using AWS CLI.
Add CodeArtifact as a repository in your Maven project (pom.xml).
Use mvn install to fetch dependencies from CodeArtifact.
Use mvn deploy to upload your libraries to CodeArtifact.


After creating Repo we will open VScode and go the file which we clones through github and checkout 
to the ci-aws branch
